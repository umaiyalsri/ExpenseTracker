# Expense Tracker
A Spring Boot Expense Tracker application for managing and tracking expenses.

## Features
- Add Expense
- Edit Expense
- Delete Expense
- Search Expense
- Category Filter
- Total Expense Calculation

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- H2 Database
- HTML/CSS
